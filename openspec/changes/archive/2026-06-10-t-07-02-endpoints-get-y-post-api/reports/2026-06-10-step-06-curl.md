### POST /api/habits (happy path)
```powershell
Invoke-WebRequest -Uri http://localhost:3001/api/habits -Method POST -ContentType "application/json" -Body '{"emoji":"🏃","name":"Test curl","pointsPerDay":10,"penalty":5}' -UseBasicParsing
```

**Respuesta:**
```json
{"id":4,"userId":1,"emoji":"🏃","name":"Test curl","pointsPerDay":10,"penalty":5,"isActive":true,"createdAt":"2026-06-10T21:15:19.283Z"}
```

**Código de estado:** 201 Created
**Verificación:** PASS — hábito persistido con `userId`, `isActive: true`

## Pruebas de casos de error

### POST name vacío (400 VALIDATION_ERROR)
```powershell
Invoke-RestMethod -Uri http://localhost:3001/api/habits -Method POST -ContentType 'application/json' -Body '{"emoji":"run","name":"","pointsPerDay":10,"penalty":5}'
```

**Respuesta:**
```json
{"code":"VALIDATION_ERROR","message":"Datos inválidos","details":[{"field":"name","message":"El nombre es obligatorio"}]}
```

**Código de estado:** 400 Bad Request
**Verificación:** PASS — `ApiErrorResponse` con `details`
