### POST /api/rewards — happy path (US-11 S1)
```bash
curl.exe -s -X POST http://localhost:3001/api/rewards \
  -H "Content-Type: application/json" \
  -d '{"emoji":"🎬","name":"Ir al cine","description":"Tarde de peli","cost":50}'
```

**Respuesta:** `{"id":3,"userId":1,"emoji":"🎬","name":"Ir al cine","description":"Tarde de peli","cost":50,"isActive":true,"createdAt":"..."}`

**Código de estado:** 201 Created

### POST /api/rewards — validación cost (US-11 S4)
```bash
curl.exe -s -X POST http://localhost:3001/api/rewards \
  -H "Content-Type: application/json" \
  -d '{"emoji":"🎬","name":"Test","description":"Test","cost":0}'
```

**Respuesta:** `{"code":"VALIDATION_ERROR","message":"Datos inválidos","details":[{"field":"cost","message":"El coste debe ser mayor que 0"}]}`

**Código de estado:** 400 Bad Request  
**Verificación:** PASS — `field: "cost"`
